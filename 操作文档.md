# 招商线索全自动爬取系统（Web版）操作文档

本系统用于：上传产业链结构 Excel（工作表名=产业链，表内“名称”列=节点），先采集原始新闻舆情，再通过 AI 穿透抽取招商线索，支持一键导出 CSV。

## 1. 运行环境
- 操作系统：Windows 10/11
- Python：3.10 或更高
- 网络：可访问搜索引擎与 DeepSeek API（需要 API Key）

## 2. 解压与目录
解压后请保持目录结构不变，进入解压目录内的 `investment_lead_system` 文件夹。

关键文件：
- main.py：Web 服务入口
- crawler_engine.py：联网检索 + AI 抽取引擎
- requirements.txt：依赖列表
- run_config.example.json：配置模板（不含密钥）
- setup_windows.bat：一键安装脚本
- start_windows.bat：一键启动脚本

## 3. 配置（必须）
### 3.1 配置 DeepSeek API Key
首次运行建议直接执行 `setup_windows.bat`，它会自动生成 `run_config.json`。

然后打开 `run_config.json`，填写：
- `deepseek_api_key`

提示：
- 不要把真实密钥提交到代码仓库、不要发给他人。
- 也可以使用环境变量：`DEEPSEEK_API_KEY`（优先级高于文件配置）。

## 4. 安装依赖（首次必须）
双击运行：
- `setup_windows.bat`

它会：
- 创建虚拟环境 `.venv`
- 安装依赖
- 生成 `run_config.json`（若不存在）

## 5. 启动系统
双击运行：
- `start_windows.bat`

启动成功后在浏览器打开：
- http://127.0.0.1:8888

停止方式：
- 直接关闭终端窗口，或在终端按 `Ctrl + C`

## 6. 使用流程
0. 先采集原始新闻（推荐用现有脚本 crawl_raw_news.py）
   - 在上级目录运行：`E:\practice\爬虫\baidu_wenxin_spider\crawl_raw_news.py`
   - 采集完成后会生成：`raw_news_leads.csv`
   - 本系统默认从 `raw_news_leads.csv` 读取新闻再穿透成线索
1. 上传产业链 Excel（示例：`成武产业链-层级版.xlsx`）
   - 工作表名：产业链名称
   - 每个工作表中：`名称` 列为节点（如“上游/中游/下游”等）
2. 输入落商地名称与 ID（如：成武 / 152），保存配置
3. 点击“启动智能挖掘”
4. 等待右侧出现“穿透挖掘成果”
5. 点击“导出CSV”一键下载结果

## 7. 导出 CSV
页面右上角点击“导出CSV”即可下载。

也可直接访问接口（浏览器打开会下载）：
- `/api/export_leads?format=csv`

## 8. 常见问题排查
### 8.1 端口被占用（WinError 10013 或 8000/8888 启动失败）
- 换端口启动：修改 `start_windows.bat` 中的 `--port 8888` 为其他端口（如 8899）
- 或结束占用端口的进程后再启动

### 8.2 一直没有线索数据（valid_leads=0）
常见原因：
- 未先生成 `raw_news_leads.csv`（需先运行 crawl_raw_news.py），或文件路径不在程序可发现的位置
- AI 输出不包含真实企业全称，被严格过滤（企业名称必须以“有限公司/股份有限公司/有限责任公司/集团有限公司”结尾）
- 当前网络无法访问新闻源/DeepSeek API

建议：
- 打开 `/api/status` 查看 `stats` 与 `logs`，确认是否出现“已加载原始新闻…（来源：raw_news_leads.csv）”
- 先从少量产业链/节点测试（Excel 表内节点不要太多）
- 更换网络环境或允许程序联网访问

### 8.3 页面 500 Internal Server Error
通常是模板/依赖/路径异常：
- 确认是否在 `investment_lead_system` 目录内启动
- 重新运行 `setup_windows.bat` 安装依赖
- 查看启动窗口中的报错堆栈定位问题

## 9. 免 Python 安装版（PyInstaller）
如果需要给非技术同事使用，推荐使用 PyInstaller 的目录版（onedir）或单文件版（onefile）。

### 9.1 目录版（推荐，启动更快）
将打包生成的文件夹完整发给同事（不要拆散文件）：
- `investment_lead_system_web/`

同事电脑运行步骤：
1. 在同一目录下创建 `run_config.json`（可复制 `run_config.example.json` 重命名）
2. 填写 `deepseek_api_key`
3. 双击运行 `investment_lead_system_web.exe`
4. 浏览器打开 `http://127.0.0.1:8888`

### 9.2 单文件版（便于分发）
将 `investment_lead_system_web.exe` 发给同事。

同事电脑运行步骤：
1. 在 exe 同目录创建 `run_config.json`（可复制 `run_config.example.json` 重命名）
2. 填写 `deepseek_api_key`
3. 双击运行 exe，浏览器打开 `http://127.0.0.1:8888`

注意：
- 单文件版首次启动会解压到临时目录，启动会比目录版慢一些
- `uploads/` 会在运行目录自动生成，用来保存上传的 Excel
